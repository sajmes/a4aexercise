An interactive step-by-step questionnaire that recommends the ideal Automattic product stack based on a visitor's site type, traffic, ecommerce needs, security requirements, and budget.

== Description ==

The Automattic Solution Builder is a guided wizard block that walks visitors through five focused questions about their web project requirements. After completing the questionnaire, it instantly displays a tailored recommendation from the Automattic product ecosystem — including WordPress VIP, Pressable, WooCommerce, and Jetpack — along with explanations and direct links to each product.

**Key features:**

* Five-step wizard with smooth transitions
* Progress indicator showing the current step
* Smart recommendation engine that evaluates answers across all dimensions
* Clean, professional card-based design with generous whitespace
* Fully interactive on the frontend with no page reloads
* Responsive layout that works on all screen sizes

**Recommendation logic covers:**

* WordPress VIP for enterprise-grade needs
* Pressable + Jetpack for mid-range projects
* WooCommerce integration when ecommerce is required
* Sensible defaults for starter projects

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/telex-automattic-solution-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the "Automattic Solution Builder" block to any post or page via the block editor.

== Frequently Asked Questions ==

= Can I customize the questions or recommendation logic? =

The questions and logic are built into the block. Future versions may add customization options via the block inspector.

= Does the block send any data to external servers? =

No. All logic runs entirely in the visitor's browser. No data is collected or transmitted.
